# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-29

### Added
- Initial release of AI Product Manager Copilot
- Core data analysis engine with sentiment analysis and theme extraction
- Recommendation engine for features, positioning, and priorities
- Feature prioritization with RICE, MoSCoW, and Value vs Effort frameworks
- Market analysis module for competitive intelligence
- User research module for interview and survey analysis
- Analytics interpreter for metrics and KPI analysis
- Roadmap planning module with scenario analysis
- Strategy development module with GTM planning
- FastAPI server with comprehensive REST API
- Web-based GUI with multiple analysis tools
- Docker and Docker Compose configuration for easy deployment
- Comprehensive documentation and deployment guides
- Unit tests for core modules
- Support for multiple LLM providers (OpenAI, Anthropic)
